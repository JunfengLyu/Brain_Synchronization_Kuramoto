Python:
- Python >= 3.10
- numpy
- scipy
- matplotlib
- pandas
- scikit-learn

MATLAB:
- Recommended: MATLAB R2021b or newer.
- Required built-ins/toolboxes used by the provided scripts:
  - MATLAB base graphics and ODE solvers
  - Statistics and Machine Learning Toolbox (`fcdf`)
  - Optimization Toolbox is recommended for nonlinear fixed-point utilities
  - Graph and Network Algorithms functions (`graph`, `distances`)

Dataset:
- `data_for_section3&4/rawdata.mat`
  - Multi-subject AAL-90 structural connectivity tensor used to construct the group consensus connectome.
- `data_for_section3&4/{CN,EMCI,LMCI,AD}/*.csv`
  - ADNI-derived structural connectivity matrices/parcellation CSV files used for the Alzheimer's disease continuum analysis.
- Current packaged dataset copy:
  - 1 MATLAB `.mat` file.
  - Four cohort folders: CN, EMCI, LMCI, AD.
  - 237 CSV files plus the raw AAL tensor after removing macOS metadata files.

Figure output:
- All scripts save figures to `Report/Figs/` as 300 dpi PNG files.
